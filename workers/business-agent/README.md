# Kiamichi Business Agent

AI-powered business listing agent built with Cloudflare's agents framework.

## Features

- Component Management - Update page components
- Content Generation - Create SEO-optimized blog posts  
- Social Media - Generate platform-specific content
- SEO Optimization - Analyze and improve rankings
- Task Scheduling - Queue tasks for future execution
- Human-in-the-Loop - All critical changes require confirmation

## Tech Stack

- Cloudflare Agents Framework
- Workers AI (Llama 3.1 8B)
- D1 Database
- R2 Storage
- React + Vite

## Development

```bash
npm install
npm run dev
npm run deploy
```

See full documentation at https://developers.cloudflare.com/agents/
